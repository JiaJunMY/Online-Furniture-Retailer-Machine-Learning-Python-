Disclaimer: Ensure that "online_furniture_retailer.csv" is in same directory/folder as the Python file.

This is a Python application for basic machine learning model. It is used for modifying CSV file with existing data and cleaned up for accurate machine learning uses.

This project consists of multiple libraries:
- NumPy (Numerical Computing)
- Pandas (Reading CSV files, manipulating data (edit, delete or add) and preprocessing)
- Scikit-learn (Machine Learning models & utilities)
- TensorFlow (For deep learning experimentation)
- MatPlotLib (Data visualization plotting)
- Seaborn (Statistical data visualization too)
